# Суулгах заавар (Chrome Web Store-гүйгээр)

[English](INSTALL.md) | **Монгол**

Энэ өргөтгөлийг нэг минутын дотор гараар суулгаж болно. Store-ийн бүртгэл,
төлбөр шаардлагагүй бөгөөд **Chrome, Edge, Brave, Opera, Arc** (Chromium
суурьтай аль ч хөтөч) дээр ажиллана.

> Эхлээд юу хийдгийг харах уу? [Шууд демо](https://temuulennibno.github.io/mongolian-grammar-checker/)-г
> суулгахгүйгээр туршиж үзээрэй.

---

## А хувилбар — Хувилбараас (release) суулгах (санал болгож буй)

1. [**Releases**](https://github.com/temuulennibno/mongolian-grammar-checker/releases)
   хуудас руу орж хамгийн сүүлийн **`mongolian-spell-checker-vX.Y.Z.zip`**-г
   татаж авна.
2. **Задлана (unzip).** Дотор нь дээд түвшинд `manifest.json` агуулсан фолдер
   гарч ирнэ. Энэ фолдер хаана байгааг санаж аваарай (жишээ нь Downloads).
3. Хөтчийнхөө өргөтгөлийн хуудсыг нээнэ:
   - Chrome / Brave / Opera: `chrome://extensions`
   - Edge: `edge://extensions`
4. **Developer mode**-г асаана (Chrome дээр баруун дээд буланд, Edge дээр зүүн
   талын цэсэнд).
5. **Load unpacked** дарж **задалсан фолдероо** (`manifest.json` агуулсан
   фолдер) сонгоно.
6. Болсон. **Үг** дүрсийг хэрэгслийн самбарт бэхлээд текст талбартай хуудас нээнэ.

## Б хувилбар — Эх кодоос суулгах

```bash
git clone https://github.com/temuulennibno/mongolian-grammar-checker.git
cd mongolian-grammar-checker
npm install
npm run prepare-dist     # dist/sw.js, dist/content.js-г бэлдэнэ
```

Дараа нь дээрх 3–6-р алхмыг дагаж, **төслийн фолдерыг** өөрийг нь сонгоно.

## Хэрхэн ашиглах вэ

- **Шууд шалгах**: аль ч текст талбар (textarea, input, эсвэл Gmail зэрэг баялаг
  засварлагч) дээр монголоор бичихэд буруу үгс **улаанаар** тодруулагдана.
  Тодруулсан үг дээр дарж саналуудаас зөвийг нь сонгож засна.
- **Попап шалгагч**: хэрэгслийн самбар дахь дүрс дээр дарж бичвэрээ хуулж шалгана.
- **Баруун товч**: хуудсан дээрх монгол бичвэрийг тэмдэглээд → **Монгол алдаа шалгах**.

Хуудас дээрх анхны шалгалт толь ачаалах хооронд ~1 секунд авах ба дараа нь
шуурхай болно. Бүх зүйл **офлайн** ажиллах тул таны бичвэр хөтчөөс гадагш
гардаггүй.

## Тэмдэглэл ба алдаа засах

- **"Developer mode-ийн өргөтгөлийг идэвхгүй болгох" сонордуулга**: Chrome нь
  unpacked өргөтгөл бүрт үүнийг харуулдаг. Хор хөнөөлгүй — хааж эсвэл үл
  тоомсорлож болно. (Зөвхөн Web Store-оос суулгасан тохиолдолд энэ
  сонордуулга гарахгүй.)
- **Зарим хуудсан дээр ажиллахгүй**: content script нь `chrome://` хуудас,
  Chrome Web Store, бусад өргөтгөлийн хуудсан дээр ажиллаж чадахгүй. Энгийн вэб
  сайт ашиглана уу.
- **Шинэчлэх**: шинэ хувилбарын zip-г татаад хуучин фолдер дээр (эсвэл шинэ
  фолдерт) задлаад, өргөтгөлийн картан дахь **↻ reload** дүрс дээр дарна. Эх
  кодоос суулгасан бол `git pull && npm run prepare-dist` хийгээд reload дарна.
- **`.crx` файл**: орчин үеийн Chrome аюулгүй байдлын үүднээс бие даасан `.crx`
  файл суулгахыг хориглодог тул дээрх **Load unpacked** арга нь дэмжигдсэн арга
  юм.
- **Хамрах хүрээ**: энэ нь **зөв бичгийн (typo)** шалгалт хийдэг, дүрмийн биш.
  Албан ёсны тольд байхгүй ярианы үгсийг алдаатай гэж тэмдэглэнэ.
